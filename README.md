# 🏏 T20 Cricket Best Playing XI Analysis Dashboard

## 📌 Project Overview

This project leverages Data Analytics and Business Intelligence techniques to identify the strongest T20 Cricket Playing XI based on player performance data from the T20 World Cup. The analysis combines Python-based data cleaning and transformation with Power BI visualizations to create an interactive dashboard that assists in selecting the most balanced and high-performing team.

The dashboard evaluates players across multiple roles, including openers, middle-order batsmen, finishers, all-rounders, wicketkeepers, and bowlers, using key cricket performance metrics.

---

## 🎯 Objectives

* Analyze T20 World Cup player performance data.
* Identify the best-performing players for each role.
* Build a data-driven Playing XI selection framework.
* Create interactive dashboards for player comparison and team selection.
* Enable users to explore player statistics through dynamic visualizations.

## 📂 Project Workflow

### 1. Data Collection

Collected T20 World Cup datasets containing:

* Match Results
* Batting Summary
* Bowling Summary
* Player Information

### 2. Data Cleaning & Transformation

Using Python and Pandas:

* Extracted data from JSON files.
* Cleaned player names and match records.
* Mapped players with corresponding match IDs.
* Created structured datasets for analysis.
* Exported transformed data into CSV files.

### 3. Data Modeling

Developed a star-schema model in Power BI with:

* Match Summary Table
* Batting Summary Table
* Bowling Summary Table
* Player Information Table

### 4. Dashboard Development

Created role-based dashboards for:

* Openers
* Middle Order Batsmen
* Finishers
* All-Rounders
* Wicket Keepers
* Bowlers
